## 🎯 适合的使用场景

| 场景                      | 建议方式                                     |
| ----------------------- | ---------------------------------------- |
| 顶部导航栏                   | `<Link>` 或 `<button>` + `useNavigate` 都可 |
| 表单提交成功后跳转               | `useNavigate`                            |
| 点击卡片跳转详情页               | `useNavigate`                            |
| 动态路径（如 `/post/${id}`）跳转 | `navigate(\`/post/\${id}\`)\`            |
